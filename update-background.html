To update your portfolio with the cosmic background:

  1. Save the cosmic image to: c:\Users\Ian\Desktop\n\assets\cosmic-nova.jpg
  2. Then update the background CSS in index-canvas.html around line 60 to:

    /* Cosmic background image */
    body::before {
      content: '';
      position: fixed;
      top: 0;
      left: 0;
      width: 100%;
      height: 100%;
      z-index: -2;
      background: url('assets/cosmic-silhouette.jpg') center/cover fixed;
      background-attachment: fixed;
      filter: blur(8px) brightness(0.4) contrast(1.1) saturate(0.9);
      pointer-events: none;
    }

    /* Canvas background with reduced opacity */
    #cosmicCanvas {
      position: fixed;
      top: 0;
      left: 0;
      width: 100%;
      height: 100%;
      z-index: -1;
      display: block;
      opacity: 0.6;
    }

    /* Artistic overlay with vignette */
    .overlay {
      position: fixed;
      inset: 0;
      z-index: -1;
      background: 
        radial-gradient(circle at 50% 50%, rgba(10,10,10,0.2), rgba(5,5,5,0.9)),
        radial-gradient(ellipse at 50% 0%, rgba(255,204,0,0.05), transparent 70%),
        radial-gradient(ellipse at 50% 100%, rgba(139,0,0,0.08), transparent 60%);
      mix-blend-mode: multiply;
      pointer-events: none;
      box-shadow: inset 0 0 100px rgba(0,0,0,0.8);
    }

What this does:

  • Uses the cosmic image as background with blur effect for artistic appeal
  • Reduces brightness/contrast for dramatic effect
  • Layers canvas starfield on top with 60% opacity
  • Adds vignette effect with golden/red glows
  • Maintains dark theme and readability
